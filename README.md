LeftIndicatorViewGroup
======================

A left indicator view group of android.

Please feel free to use this.

# Features

* Simple Left Indicator. support custom icon.

# How do I use it?

## Setup

#### Gradle

```groovy
dependencies {
  // jCenter
  compile 'com.carlos.widget:leftindicator:0.0.+'
}
```
